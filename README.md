# PDF App React

## Features
- Table editor
- PDF export
- JSON/CSV export and import
- Print support

## Installation Instructions
To get started, clone the repository and install the dependencies:

```bash
git clone https://github.com/your-username/pdf-app-react.git
cd pdf-app-react
npm install
```

## Build Instructions
To build the project for production, run:

```bash
npm run build
```

## Usage Guide
### Creating Tables
To create tables, follow these steps:
1. Go to the tables section in the app.
2. Use the table editor to add rows and columns.

### Exporting Data
To export data to PDF, JSON, or CSV format, use the export options available in the app.

### Importing JSON
To import JSON data, select the import option and upload your JSON file.

### Printing
Use the print support feature to print your tables directly from the app.
